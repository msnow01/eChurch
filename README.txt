Setup Instructions
- Create a connection.php file, use the connection.example.php file as a reference
- Create a config.php file, use the config.example.php file as a reference

Admins have the option to upload audio, embed videos, and publish notices, alerts or resources through the dashboard, in addition to managing all users, generating user reports, uploading files and editing categories. Each new user must sign up and be given distinct permissions by admins before their account becomes active. This site can easily be customized with different colors and branding through the admin console!
