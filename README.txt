Admins have the option to upload audio, embed videos, and publish notices, alerts or resources through the dashboard, in addition to managing all users, generating user reports, uploading files and editing categories. Each new user must sign up and be given distinct permissions by admins before their account becomes active. This site can easily be customized with different branding, colors and logos! 

*Note that for privacy of personal information, this project is a duplicate of the original website developed with placeholder content.
